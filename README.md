# merge-sort-benchmark

An algorithmic performance analysis tool built to demonstrate the efficiency of the **Divide and Conquer** paradigm through a custom Merge Sort implementation.

## 🚀 Project Vision
This project is part of the **Engine Series**, focusing on the intersection of algorithmic logic and performance benchmarking. It provides a robust Python-based suite to visualize the performance gap between $O(n \log n)$ and $O(n^2)$ sorting strategies.

## 🛠 Features
- **Recursive Merge Sort**: A hand-crafted implementation of the merge sort algorithm.
- **Timing Decorators**: High-order functions to measure execution time with microsecond precision.
- **Data Generation**: Utilities to generate random, sorted, and reverse-sorted datasets.
- **Performance Analysis**: A benchmarking dashboard to compare results.

## 📁 Repository Structure
- `merge_sort.py`: Core algorithmic logic (Divide and Conquer).
- `benchmarker.py`: Timing decorators and benchmarking execution.
- `requirements.txt`: Project dependencies (if any).
